# Vagrant LAMP Environment
Simple PHP7 Vagrant Environment 

## Technology included
* [Ubuntu 16.04/xenial]
* [Apache2.4](http://nginx.org/)
* [MySQL 5.7]
* [PHP 7]
## Requirements
Cpus: 2Cores
Ram: >=4GB
## Running

Home Directory map: ./data/www -> /var/www/html
Mysql User:
   * root/root (localhost)
   * dbuser/123456 (localhost & %)
